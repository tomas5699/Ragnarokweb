Acerca de Far Cry 4 para PC
Escondido en la cordillera del Himalaya se encuentra Kyrat, un país sumergido en la tradición y la violencia.
Durante el juegoa seras Ajay Ghale con el viajarás a Kyrat para satisfacer la ultima voluntad de tu madre, pero cuando llegas notaras que has entrado en una guerra civil provocada para finalizar con el régimen opresivo del dictador Pagan Min. Explora y navega dentro de este brutal juego, en el que el peligro y los imprevistos están tan cerca que tu ni te lo imaginas. Cada decisión que tomes aquí cuenta, y cada segundo es una historia. Bienvenido a Far Cry 4.



Requerimientos de Sistema

Mínimos:
Sistema Operativo: Windows 7 SP1 de 64 Bits
Procesador: Intel Core i5-750 de 2.6 GHz o un AMD Phenom II X4 955 de 3.2 GHz
Memoria: 4 Gigas de RAM
Gráficos: NVIDIA GeForce GTX 460 o una AMD Radeon HD5850 (con 1 Giga de VRAM)
DirectX: Versión 11
Espacio en el Disco: 30 Gigas disponibles para la instalación


Recomendados:

Sistema Operativo: Windows 7 | Windows 8.1 de 64 Bits
